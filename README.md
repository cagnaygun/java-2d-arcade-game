# 🚀 GalaxyBlast

A fast-paced 2D arcade space shooter game developed in Java.

## 🎮 Features

- Smooth player movement
- Meteor enemy system
- Boss fight mechanics
- Sound effects and arcade atmosphere
- Health & pickup system
- Projectile shooting system
- Score tracking
- Retro arcade-inspired gameplay

---

## 🕹 Controls

| Key | Action |
|---|---|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
| SPACE | Shoot |

---

## 🛠 Technologies Used

- Java
- Java Swing / Java2D
- Object-Oriented Programming
- Game Loop Architecture
- Observer & Singleton Patterns
